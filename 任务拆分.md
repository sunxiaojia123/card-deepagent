### 任务 1.5 — SSE 流式适配器（Text）

| 项目 | 内容 |
|------|------|
| **目标** | 将 agent 的 `messages` stream 转换为统一 SSE 协议 `event=text` |

| **验收** | curl SSE 端点，逐 token 收到 text 事件，最终收到 done |

| **验收** | 启动服务 → curl 发消息 → SSE 流式返回 text + done |
