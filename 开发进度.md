## 踩坑记录

| 问题 | 解决 |
|------|------|
| Postgres checkpointer 的 `aget_tuple()` 无法加载 messages（blob 版本不匹配） | 改用 `agent.aget_state(config)` 读取历史 |
| psycopg 默认不 autocommit，`init_db()` CREATE TABLE 被回滚 | `conn.set_autocommit(True)` |
| `astream` 单 stream_mode 返回 2-tuple `(mode, data)`，非文档中的 3-tuple `(namespace, mode, data)` | 适配器按 `(mode, data)` 解析 |
| MCP ToolNode 不注册 tool → `_validate_tool_call` 返回 error | `awrap_tool_call` 拦截 `request.tool is None` → `replace(request, tool=real_tool)` → handler 正常执行 |
| `langchain_mcp_adapters` 每 tool call 创建新 session（开销大） | 接受现状；session 由 `async with create_session` 自动管理，无泄漏 |
| SSE `event: status` 检测 `tool_call_chunks` 不适用于所有模型 | 同时检查 `tool_calls`（完整 tool call）和 `tool_call_chunks`（流式） |
